# **App Name**: Aether Settings Sync

## Core Features:

- Settings Display: Display user-configurable settings such as theme and language.
- Theme Management: Allows users to switch between light and dark themes and persist the selected theme across sessions via DataStore.
- Language Selection: Lets users select their preferred language and stores the selection using DataStore.
- Data Persistence: Utilizes DataStore to persist user settings, ensuring they are restored on app restart.
- User Profile Integration: Fetch and display the username and avatar from Datastore to show in main and settings fragment.
- Navigation Drawer: Implement a navigation drawer component allowing seamless movement between app sections and displaying profile info.
- Logout Handling: Add Firebase logout functionality via the navigation drawer to ensure secure session management.

## Style Guidelines:

- Primary color: Soft blue (#A0CFEC) to reflect calmness and reliability, aligning with settings and personalization.
- Background color: Very light blue (#F0F8FF), near-white, creating a clean, uncluttered appearance that ensures readability and focus on content.
- Accent color: Light teal (#77D8D8), offering subtle contrast while maintaining a harmonious and unintrusive aesthetic.
- Headline font: 'Poppins' (sans-serif), for headlines and UI elements. Body font: 'PT Sans' (sans-serif) if blocks of text longer than a paragraph are required.
- Use minimalistic, clear icons for settings and navigation items.
- Maintain a clean and intuitive layout, leveraging Android's Material Design principles.
- Use subtle transitions for menu navigation and settings changes.
